# INSTRUCCIONES DE INSTALACION RAPIDA

## PASO 1: DESINSTALAR VERSION ANTERIOR (si existe)

1. Abri "Panel de Control" > "Programas y caracteristicas"
2. Busca "GEPCP Ferreteria El Pana"
3. Click derecho > Desinstalar
4. Espera a que termine

## PASO 2: INSTALAR NUEVA VERSION

1. Busca el archivo:
   `GEPCP Ferreteria El Pana\Instalador\Output\GEPCP_FerreteriaElPana_Setup_v1.0.0.exe`

2. Click derecho > "Ejecutar como administrador"

3. Sigue el asistente de instalacion

4. Marca la opcion "Crear acceso directo en el escritorio" (opcional)

## PASO 3: INICIAR EL SISTEMA

Opcion A: Doble click en el acceso directo del escritorio
Opcion B: Menu Inicio > "GEPCP Ferreteria El Pana"

## QUE ESPERAR AL INICIAR

1. Se abre una ventana de CONSOLA NEGRA (NO LA CIERRES, es normal)
   - Esta consola muestra los logs en tiempo real
   - Debe decir "Now listening on: http://localhost:XXXX"

2. Se abre el SPLASH SCREEN en el navegador
   - Muestra el logo de Ferreteria El Pana
   - Dice "Iniciando sistema..."
   - Automaticamente te redirige al sistema

3. Se abre el SISTEMA en el navegador

## SI HAY PROBLEMAS

### El sistema no inicia
1. Ejecuta "Reparar Sistema" desde Menu Inicio
2. Espera 30 segundos
3. Revisa la consola de logs

### No llega el email de recuperacion
1. Revisa la consola, debe decir:
   "✓ Codigo enviado exitosamente a: tu@email.com"
2. Revisa la carpeta de SPAM
3. Espera hasta 2 minutos

### Conflicto de puerto
- El sistema detecta automaticamente un puerto libre
- Si el puerto 5001 esta ocupado, usa 5002, 5003, etc.
