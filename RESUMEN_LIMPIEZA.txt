## 🚀 INSTRUCCIONES DE USO

### Para Instalar
1. Ejecutar `Setup_GEPCP_FerreteriaElPana.exe` como administrador
2. Seguir el asistente de instalación
3. Al finalizar, la aplicación se inicia automáticamente
4. El sistema abrirá el navegador en http://localhost:5002

### Para Usar
1. **Primer acceso**: Usar usuarios predeterminados
   - Usuario RRHH: `admin.rrhh` / `Pana2024`
   - Usuario Jefatura: `jefatura` / `Pana2024`

2. **Acceso posterior**: Hacer clic en el acceso directo del escritorio

3. **Si se cierra el navegador**: El sistema sigue corriendo en segundo plano, solo abrir el acceso directo nuevamente

### Para Desinstalar
1. Panel de Control → Programas y características
2. Buscar "GEPCP Ferreteria El Pana"
3. Clic en Desinstalar
4. El proceso detiene la aplicación automáticamente
