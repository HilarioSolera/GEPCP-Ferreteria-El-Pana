📋 RESUMEN DE CAMBIOS REALIZADOS
═════════════════════════════════════════════════════════════════════════════

✅ 1. SCRIPT DE INICIO AUTOMÁTICO
   Archivo: IniciarSistema.bat

   Funcionalidad:
   • Verifica si el servidor ya está corriendo
   • Si está activo → lo apaga → lo reinicia
   • Si está inactivo → lo inicia directamente
   • Abre navegador automáticamente
   • Espera máximo 20 segundos

   Ventaja: Sin necesidad de monitor de inactividad o heartbeat

✅ 2. INSTALADOR PROFESIONAL (Inno Setup)
   Archivo: GEPCP.iss

   Funcionalidad:
   • Interface profesional y amigable
   • Instalación automática en C:\Program Files\
   • Crea accesos directos (Escritorio + Menú Inicio)
   • Validación de requisitos (.NET 8, curl)
   • Desinstalación limpia

   Resultado: GEPCP-Ferreteria-El-Pana-Setup.exe (~50-100 MB)

✅ 3. ENDPOINTS DE CONTROL
   Archivo: Controllers/ServerController.cs

   Endpoints:
   • GET/POST  /api/server/ping      → Verificar estado
   • POST      /api/server/shutdown  → Apagar servidor

   Ventaja: Accesibles sin autenticación desde el script .bat

✅ 4. SCRIPT DE COMPILACIÓN
   Archivo: build-installer.bat

   Funcionalidad:
   • Compila en Release automáticamente
   • Ejecuta Inno Setup automáticamente
   • Genera Setup.exe en carpeta Output/
   • Validación de requisitos (Inno Setup)

✅ 5. DOCUMENTACIÓN COMPLETA
   Archivos:
   • INSTALACION.md              → Guía para usuarios finales
   • COMPILAR-INSTALADOR.md      → Guía para desarrolladores
   • README-SETUP.txt            → Este archivo

🚀 GENERAR INSTALADOR EN 3 PASOS
═════════════════════════════════════════════════════════════════════════════

1. PREREQUISITOS (si no tienes):
   ✓ Descargar Inno Setup 6: https://jrsoftware.org/isdl.php
   ✓ Instalar normalmente
   ✓ Verificar que curl está en el sistema

2. GENERAR:
   Opción A - Automático (RECOMENDADO):

   Navega a la carpeta del proyecto y ejecuta:

	   build-installer.bat

   ¡Listo! El Setup.exe se genera automáticamente.

   Opción B - Manual:

   Terminal de Desarrollador de Visual Studio:

	   dotnet build --configuration Release --no-restore
	   "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" GEPCP.iss

3. RESULTADO:

   Output\GEPCP-Ferreteria-El-Pana-Setup.exe

   Este es el archivo a distribuir a usuarios

📦 FLUJO DE USUARIO
═════════════════════════════════════════════════════════════════════════════

INSTALACIÓN (Ejecuta usuario):

   1. Doble clic en GEPCP-Ferreteria-El-Pana-Setup.exe
   2. Sigue los pasos del asistente
   3. Acepta permisos de administrador si se pide
   4. ¡Listo! Se instala y abre automáticamente

USO DIARIO (Ejecuta usuario):

   1. Doble clic en acceso directo "GEPCP Ferretería El Pana"
   2. El script verifica/reinicia el servidor
   3. Se abre navegador en http://localhost:5002
   4. ¡Sistema listo para usar!

CERRAR SISTEMA (Usuario elige):

   • Opción 1: Botón "Cerrar Sistema" en la app
   • Opción 2: Cerrar navegador (servidor sigue corriendo)
   • Opción 3: Ctrl+C en ventana del script

💡 TIPS ÚTILES
═════════════════════════════════════════════════════════════════════════════

CAMBIAR PUERTO (si necesitas):
   • Editar IniciarSistema.bat: set SERVER_URL=http://localhost:5003
   • Editar launchSettings.json del proyecto
   • Editar GEPCP.iss si es necesario

CAMBIAR ICONO:
   • Reemplazar logo en SetupIconFile de GEPCP.iss
   • Usar archivo .ico o .jpg

VERSIONES:
   • Actualizar AppVersion en GEPCP.iss para nuevas versiones
   • El script detectará cambios automáticamente

DISTRIBUCIÓN:
   • Subir Setup.exe a servidor o GitHub Releases
   • Usuarios descargan y ejecutan
   • Instalación automática
