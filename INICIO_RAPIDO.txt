📦 INSTALACIÓN
═══════════════════════════════════════════════════════════════════

1. Ubicar el instalador:

4. ✅ ¡Listo! El sistema se inicia automáticamente

🔑 ACCESO POSTERIOR
═══════════════════════════════════════════════════════════════════

1. Hacer doble clic en el acceso directo del escritorio:
   🖥️ GEPCP Ferreteria El Pana

🛠️ SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════

❌ No abre el navegador:
   → Abrir manualmente: http://localhost:5002

❌ Mensaje "No se puede conectar":
   → Verificar que la aplicación esté corriendo
   → Buscar "GEPCP Ferreteria El Pana" en Administrador de tareas
   → Si no está, ejecutar desde Menú Inicio

❌ Error de base de datos:
   → La base de datos se crea automáticamente al primer inicio
   → Esperar unos segundos y recargar la página
