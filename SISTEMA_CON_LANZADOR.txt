═══════════════════════════════════════════════════════════════════
📦 COMPONENTES DEL SISTEMA
═══════════════════════════════════════════════════════════════════

1. 🚀 FERRETERIA EL PANA.EXE (Lanzador)
   - Archivo ejecutable con el ícono de la ferretería
   - Verifica si el servidor está corriendo
   - Si no está corriendo, lo inicia automáticamente
   - Abre el navegador en http://localhost:5002
   - Es el acceso directo principal del usuario

2. 🖥️ GEPCP FERRETERIA EL PANA.EXE (Servidor)
   - El servidor web ASP.NET Core
   - Se ejecuta en segundo plano
   - Puerto fijo: 5002
   - Se inicia automáticamente con Windows (opcional)

═══════════════════════════════════════════════════════════════════
🔧 CÓMO FUNCIONA
═══════════════════════════════════════════════════════════════════

ESCENARIO 1: Primera vez / Servidor apagado
────────────────────────────────────────────
Usuario hace doble clic en "Ferreteria El Pana.exe"
   ↓
Lanzador verifica puerto 5002
   ↓
Puerto cerrado → Lanzador inicia "GEPCP Ferreteria El Pana.exe"
   ↓
Espera hasta que el servidor responda (máx 30 segundos)
   ↓
Abre el navegador en http://localhost:5002
   ↓
✅ Usuario ve la página splash y luego el login

ESCENARIO 2: Servidor ya corriendo
───────────────────────────────────
Usuario hace doble clic en "Ferreteria El Pana.exe"
   ↓
Lanzador verifica puerto 5002
   ↓
Puerto abierto → El servidor ya está corriendo
   ↓
Abre directamente el navegador en http://localhost:5002
   ↓
✅ Usuario ve la página splash y luego el login

ESCENARIO 3: Usuario cierra el navegador
─────────────────────────────────────────
Usuario cierra Firefox/Chrome/etc
   ↓
El servidor sigue corriendo en segundo plano
   ↓
Usuario vuelve a hacer doble clic en el acceso directo
   ↓
Lanzador detecta que el servidor ya está corriendo
   ↓
Abre nuevamente el navegador
   ↓
✅ Acceso instantáneo (sin espera)

🚀 INSTRUCCIONES DE USO
═══════════════════════════════════════════════════════════════════

INSTALACIÓN:
1. Ejecutar Setup_GEPCP_FerreteriaElPana.exe como administrador
2. Seguir el asistente
3. Marcar las opciones:
   ☑ Crear ícono en el escritorio
   ☑ Iniciar servidor automáticamente con Windows
4. Finalizar instalación
5. El sistema se abrirá automáticamente

USO DIARIO:
1. Si marcaste auto-inicio: Solo hacer doble clic en el acceso directo
2. Si NO marcaste auto-inicio: 
   - Primera vez del día: Hacer doble clic (esperar 10-30 segundos)
   - Siguientes veces: Hacer doble clic (acceso instantáneo)

CERRAR EL SISTEMA:
- Opción 1: Dejar el servidor corriendo todo el día (recomendado)
- Opción 2: Administrador de tareas → Finalizar "GEPCP Ferreteria El Pana.exe"
- Opción 3: Reiniciar la computadora

═══════════════════════════════════════════════════════════════════
🔧 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════

❌ "No se pudo iniciar el servidor"
   → Verificar que todos los archivos estén en:
	 C:\Program Files\GEPCP Ferreteria El Pana\
   → Reinstalar si faltan archivos

❌ "Error al abrir el navegador"
   → El servidor está corriendo
   → Abrir manualmente: http://localhost:5002

❌ Firefox dice "No se puede conectar"
   → El servidor aún no ha iniciado
   → Esperar 10-30 segundos
   → O verificar en Administrador de tareas si está corriendo

❌ El acceso directo no hace nada
   → Ejecutar como administrador
   → Verificar que el archivo exista:
	 C:\Program Files\GEPCP Ferreteria El Pana\Ferreteria El Pana.exe
