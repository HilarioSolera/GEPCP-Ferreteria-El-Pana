¿DÓNDE ESTÁ EL .EXE Y CÓMO USARLO?
═════════════════════════════════════════════════════════════════════════════

El instalador (.exe) se genera en:

	Output\GEPCP-Ferreteria-El-Pana-Setup.exe

Tamaño: ~50-100 MB

---

¿CÓMO INCLUIR EL LOGO EN TODO LADO?
═════════════════════════════════════════════════════════════════════════════

✅ OPCIÓN 1: AUTOMÁTICO (Recomendado)

Ejecuta este comando UNA SOLA VEZ:

	build-complete.bat

Este script hace TODO automáticamente:
1. Convierte logo-el-pana.jpg → GEPCP.ico
2. Compila la aplicación
3. Genera el instalador
4. Configura el logo en todo lado

¡El .exe estará listo en Output\!

---

✅ OPCIÓN 2: PASO A PASO

1. Convertir Logo:
   - Opción A (Python):
	 pip install Pillow
	 python convert-to-ico.py

   - Opción B (ImageMagick):
	 convert-logo-to-ico.bat

   - Opción C (En línea):
	 Ir a: https://convertio.co/jpg-ico/
	 Subir: wwwroot\images\logo-el-pana.jpg
	 Descargar: GEPCP.ico → guardar en raíz del proyecto

2. Compilar:
   dotnet build --configuration Release --no-restore

3. Generar Instalador:
   "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" GEPCP.iss

Resultado: Output\GEPCP-Ferreteria-El-Pana-Setup.exe

---

SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════════════════════════════════════════

❌ "build-complete.bat no funciona"
→ Asegúrate que estás en la carpeta correcta
→ Ejecuta como Administrador

❌ "Python no encontrado"
→ Descargar: https://www.python.org/downloads/
→ Instalar y seleccionar "Add Python to PATH"

❌ "Inno Setup no encontrado"
→ Descargar: https://jrsoftware.org/isdl.php
→ Instalar en carpeta predeterminada

❌ "No se puede convertir imagen"
→ Verificar que wwwroot\images\logo-el-pana.jpg existe
→ Probar conversión en línea: https://convertio.co/jpg-ico/
